Featured Posts Widget shows by selected categories


== Description ==

CCR Featured Posts Plugin developed by [CodexCoder](http://www.codexcoder.com/ "CodexCoder").

Don't need an extra configurations. Drag and drop CCR Featured Posts widget into your sidebar. Choose a category to 

show Featured Posts into your sidebar.


== Installation ==

1. Upload 'ccr-featured-posts' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag and Drop 'CCR Featured Posts' widget into your sidebar and select a category to show Featured Posts by Category.
